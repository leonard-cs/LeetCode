## Difference Array (Difference Technique)
[Neetcode Video](https://youtu.be/iOm-VdQJmSs?si=ThRfHJcR6CQTOsus&t=568)

Difference array algorithms are useful for efficiently handling range update problems.
The idea behind a difference array is to compute the difference between consecutive elements of an array, which allows you to perform range updates in constant time and later reconstruct the updated array by performing a prefix sum.

## Monotonic Stack
Youtube videos: [AlgoMonster](https://youtu.be/Dq_ObZwTY_Q?si=NeCJ3ZaBwHJ5Wn2p), [AlgoMasterIO](https://youtu.be/DtJVwbbicjQ?si=wcwUUErlAEYa4gxH), [CodeLucy](https://youtu.be/15OuNilu0r0?si=nAGqc_8Olew829AP)

The Monotonic Stack pattern uses a stack to maintain a sequence of elements in a specific order (increasing or decreasing).

Use this pattern for problems that require finding the **next greater** or **smaller** element. [AlgoMasterIO](https://blog.algomaster.io/i/146649626/monotonic-stack)
